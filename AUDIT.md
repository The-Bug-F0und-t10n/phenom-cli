## Conventions

- **External refs** counts files (other than the defining file) that mention the symbol via `grep -w`. Includes import statements, call sites, and incidental identifier mentions — overcounts slightly but never undercounts.
- **Status: dead?** flags exports with 0 external refs. NOT all are truly dead: tool handlers wired through string-keyed registrars, event listeners attached by event name, and test-only fixtures may legitimately show 0 grep hits.
- **Tests** match each source file against `src/tests/**/test-*.ts` by stem token-set overlap (≥ N-1 tokens must coincide) or substring. Files showing 'none located by name match' may still be covered through integration tests that exercise the larger pipeline.
- **Purpose** is extracted from the file's first comment block (top of file, before any code). Files without a leading comment show `_(no header comment)_`.
