═══════════════════════════════════════════════════════════════

1. JSON parsing quebrava com código contendo '{' em strings
2. Shell injection em run_code (vulnerabilidade crítica)
3. Deliberation gaps silenciosos corrompendo memória
4. Conteúdo vazio skipado sem notificação
5. AutoExpandCreateFiles adicionando arquivos irrelevantes
6. Modelo 7B aceitando placeholders e código trivial
7. Mensagens de erro genéricas em retry loops
8. Ausência de diretrizes de clean code
