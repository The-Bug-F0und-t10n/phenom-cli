## Insights
- 9B models degrade when prompt carries too many redundant instructions/tool descriptions.
- Session continuity should be injected safely as `current focus`, not full backlog.
- Session recall is an operational contract: the model should use `search_session` with model-chosen terms and `scope=current|all` or `session?` when prior conversation context is needed.
- Distilled memory is only useful when written to sections that are injected into prompts.
- Use DuckDuckGo as the default search engine for web context queries
- Format tool searches as 'Web context for: [query]' to maintain consistency
